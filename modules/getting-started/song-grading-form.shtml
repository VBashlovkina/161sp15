CSC 161 Module 0 Project: Program a Song

Grading Form Name(s):

This grading form expands the Program Style Summary and Checklist that applies to all program development in CSC 161.

This project will be worth 25 points, based on the following rubric:

  • program plays song/melody (5 points)






  • song/melody introduced using eSpeak (2 points)






  • song/melody is at least 30 seconds long (5 points)






  • program compiles, links, and runs (4 points)






  • description of the program (4 points)






  • creativity in making an interesting application (5 points)
